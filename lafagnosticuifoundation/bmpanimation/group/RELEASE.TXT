BmpAnim Group Release Notes
~~~~~~~~~~~~~~~~~~~~~~~~~
This component consists of two DLL's, a shared-library which provides the
component's API (BmpAnim.DLL), and a polymorphic DLL which plugs into the
window-server's "WSANIM" interface (BmpAnimSrv.DLL). 
It is able to display a series of bitmap frames at different times and position.
